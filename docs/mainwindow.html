Lingua Documentation - Main Window
A Multilingual Translation Program

The main Lingua window has two parts: the controls, and the output.

The Word box is where you type the word you want to translate. The menu to the right holds the last few words you queried for (the exact number it holds can be configured in the Preferences menu). Selecting an option from the Word menu automatically searches for it.

The Language pop-up menu selects the language to translate to or from.

The Translate button will translate the word once something has been typed into the Word box.

Note: If a translation is in progress, the "Translate" button becomes a "Stop" button. Pressing this stops the current search.

The Search For options allow you to translate to words containing the word in the box (which will return many results) or just the exact word (which significantly slims down the results).

The Translate options select which way the word will be translated. The output will switch between English being first and the Foreign word being first, depending on the option (this can be overridden in the Preferences window).

If there is a valid word in the Word box, the word can optionally be automatically be defined if you change language in the Language pop-up menu, pick a word from the Word pop-up menu, or switch either of the Search For or Translate options. This can be configured in the Preferences window.

The title bar at the top changes to indicate a search in progress or the total results returned.
