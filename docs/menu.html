Lingua Documentation - Menu
A Multilingual Translation Program

File Menu

New Window - Opens another Lingua window
Save Results... - Saves current results to a text file, with StyledEdit styles
Dictionary Updates... - Opens the Update Dictionaries window

Page Setup... - Opens the Page Setup configuration panel for your printer
Print - Prints the current results

About Lingua - Shows the super-snazzy About box

Close - Closes the current Lingua window
Quit - Closes all Lingua windows

Edit Menu

Copy - Copies text
Paste - Pastes text
Select All - Selects all the output text

Preferences... - Opens the Preferences menu

Translations

Word Wrap - Turns word wrapping on and off
Clear Results - Clears all the definition results

Help

Documentation... - Opens the documentation in the default browser

Lingua Website - Opens the default browser to the Lingua website
BeBits Page - Opens the default browser to the Lingua page on BeBits
