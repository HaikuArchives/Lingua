Lingua Documentation - Deskbar Replicant
A Multilingual Translation Program

The Lingua Deskbar replicant provides a convenient way to access the program, and execute a few extra features. The replicant can be enabled in the Preferences menu. It is automatically removed when Lingua closes, and reappears when Lingua is run again.

Right-clicking on the icon opens a new window. Left-clicking opens a menu with four options:

New Window opens another Lingua window.

Move Windows Here moves all open Lingua windows into the current workspace.

Translate Clipboard takes whatever text is currently in the clipboard, inserts it into the first open window in the current workspace, and automatically translates it.

Stay On Top makes all Lingua windows floating, meaning they appear on top of all other windows. Note: when floating, Lingua windows appear in all workspaces and aren't listed in the Deskbar.
